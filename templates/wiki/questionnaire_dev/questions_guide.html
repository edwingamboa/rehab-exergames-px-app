{% include "wiki/term_mention.html" with term=term|get_definition:"researcher" verbose="Researchers"%} should consider the following recommendations when composing questionnaire question:

  1. Involve, if possible, representatives of target respondents in the design process of a questionnaire [1].
  2. Include explanations or examples when asking about abstract things [1,2]. However, specific and concrete wording is preferred over abstract wording [3].
  3. Use simple vocabulary that respondents can understand, considering that some words have different meanings in academia and in respondents' daily life [1,4,3].
  4. Avoid complex routing for {% include "wiki/term_mention.html" with term=term|get_definition:"screening question" verbose="screening questions"%} since that reduce respondents motivation and lead to indiscriminate answering [1].
  5. Pay careful attention to demographic data that is intended to be collected. Avoid asking sensitive and threatening data. Pre-testing questionnaires allow identifying this issue [1]
  6. Use, if possible, official translators or translation services when developing questionnaires in more than one language [1].
  7. Avoid excluding disadvantaged groups of respondents to avoid biases, misunderstandings or ambiguity[1].
  8. Researchers should use specific adverbs instead of frequently, regularly, commonly, usually, hardly, etc. [5,3].
  9. Offer enough answer options when using {% include "wiki/term_mention.html" with term=term|get_definition:"closed question" verbose="closed questions"%} to avoid frustration [5]. Also, answer options should be mutually exclusive [3].
  10. Plan carefully when using {% include "wiki/term_mention.html" with term=term|get_definition:"open question" verbose="open questions"%}, such plan should include time, skills and resources required for the design and analysis processes of a questionnaire[5].
  11. Avoid starting questionnaires with threatening questions [4].
  12. Include simple instructions [4].
  13. Be as brief as possible [4].
  14. Ask one question at a time [4,3].
  15. Include easy to answer questions at the beginning of a questionnaire [2,3]. Initial questions should also address questionnaire topic directly [3].
  16. Order questions in a way that one lead naturally to the other. Additionally, researchers should be group questions on the same topic [2,3].
  17. Include sensitive questions at the end of a questionnaire to avoid respondents cutting off before important information is collected [2,3].
  18. Allow filtering questions to avoid that respondents answer questions that do not apply to them [3].
  19. Include a balance number of answer options when using rating scales. 7 options have been proved to lead to reliable and valid questionnaires [3].
  20. Label points of rating scales. Points should be divided into approximately equal units [3].
  21. Assure anonymity to avoid social desirability response bias [3].
  22. Avoid burdening respondents. To achieve this, researchers should reduce reference period, use yes/no questions instead of all that apply questions. Also, respondents shouldn't perform computations that researchers can perform.

References
